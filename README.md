# ⚔️ Realm Ascension

A deep, zone-based incremental game with discovery mechanics and strategic resource management.

## 🎮 Core Gameplay

**Realm Ascension** is an incremental game that emphasizes:
- **Discovery**: Unlock new zones by gathering resources
- **Resource Management**: Multiple resources with different acquisition methods
- **Strategic Building**: Construct buildings that produce resources
- **Unique Mechanics**: Each zone has different gameplay mechanics
- **Progression**: Advance through zones, each with new challenges

## 🗺️ Zones

1. **Grasslands** 🌾 - Starting zone. Click to gather grass and wood.
2. **Forest** 🌲 - Harvest wood efficiently with woodcutters.
3. **Mountains** ⛰️ - Mine ore and stone with specialized workers.
4. **Caves** 🕳️ - Delve for crystals and treasures.
5. **Lost Temple** 🏛️ - Commune with ancient essence.

## 🏗️ Buildings

Each zone has unique buildings:
- **Woodcutter** - Passive wood production
- **Stone Mason** - Stone quarrying
- **Miner** - Ore extraction
- **Crystal Hunter** - Crystal gathering
- **Priest** - Essence channeling

## ⚡ Upgrades

Permanent improvements that enhance your production:
- Improved Gathering
- Better Tools
- Ancient Knowledge

## 🎯 Goals

1. Gather 50 wood to unlock the Forest
2. Collect 200 wood + 100 stone to unlock the Mountains
3. Obtain 150 ore + 300 wood to unlock the Caves
4. Get 100 crystals + 300 ore to unlock the Temple
5. Reach 500 essence to complete the game

## 📊 Features

- **Persistent Save System**: Your progress is automatically saved
- **Zone Navigation**: Switch between zones to discover new mechanics
- **Resource Tracking**: Monitor production rates in real-time
- **Strategic Building**: Choose which buildings to invest in
- **Long-term Progression**: Multiple sessions of gameplay

## 🚀 Getting Started

1. Open `index.html` in a web browser
2. Click the icon in the center to gather resources
3. Build structures to increase production
4. Unlock upgrades
5. Discover new zones

## 💾 Saving

Your game automatically saves every action. Use the "Save" button to manually save, or "Reset" to start over.
